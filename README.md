# 简介
该项目是后端[faster-template-backend-with-nestjs](https://github.com/heifengli001/faster-template-backend-with-nestjs)的前端实现

关于后台服务已有能力，可前往查看查看已支持的功能

# 技术栈
+ Vue3 + vue-router + pinia
+ TypeScript
+ UI: Arco

# 已实现能力
+ 登录
+ 用户信息修改
+ 文章增删改查
    + 富文本
        + 媒体上传
    + 评论(树)
    + 点赞
+ 分类树
    + 支持排序
+ 素材管理
    + 瀑布流
    + 选择器
+ 文件上传
+ 安全
    + csrf
    + xss
